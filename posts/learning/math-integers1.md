Integers are whole numbers that can be positive, negative, or zero. They are represented on a number line like this:

$$\{..., -3, -2, -1, 0, 1, 2, 3, ...\}$$

The absolute value of an integer is its distance from zero. For example:

$$|{-5}| = 5$$

And a simple equation using integers:

$$x + 3 = 7 \implies x = 4$$
