**A review of addition**

Addition, at its core, its just the asking ourselves of the simple question _"I have some of this thing, and some of that thing, how many things do I have?"_. The easiest way we can understand this concept, is visually, through the use of a number line.

To illustrate, let's work through the following example.

> I have 3 apples and 4 pears, how many fruits do I have?

For simplicity sake, let's draw a number line and put a character ^ to symbolize the place on the line where we should start.

First, we'd draw our character on the number three, since we know we are starting with three apples. Its just that we want to know how many total fruits we have after counting the pears.

```
←───|───|───|───^───|───|───|───|───|───→
    0   1   2   3   4   5   6   7   8
```

Now, all we have to do is figure out what our result is if we were to add the pears to our apples. Adding, in terms of the number line, simply means moving a specific number of unit lenghts to the right equal to the quantity we've been told to add. (As you'll remember, a unit lenght is simply the distance from one number to another in the number line.)

So, let us use the character > to represent _one_ move to the right, and, let us use the _second_ apperance of the character ^ as our end result.

```
←───|───|───|───^───|───|───|───^───|───→
    0   1   2   3 > 4 > 5 > 6 > 7   8
```

Notice how we moved four unit lenghts to the right (four times the size of a unit lenght, or four jumps from one number to the next) and our ending place is the number seven.

This means, if we have three apples and four pears we have seven fruits total.

We can follow the same pattern if we wanted to find the sum of any numbers using a number line.

Now that we intuitively understand the concept of addition, we can learn about some of the properties that it has.

First, addition has what we call the commutative^[Commutative means *relating to or involving substitution or exchange*, as you see, the property is all about the exhanging of the places of numbers in an equation] property. That is, the order in which numbers appear on a sum will not change its result.

This means

> If A and B represent whole numbers, then
>
> A + B = B + A

Second, addition has an associative property. That is, when presented with a sum of numbers, no matter how you decide to group them, the answer will not change.

So

> If A, B and C are whole numbers, then
>
> (A + B) + C = A + (B + C)

Finally, we say that addition has the additive identity. That is, any whole number that is added to _zero_ will result in the same number.

This means

> If A is any whole number
>
> A + 0 = A
