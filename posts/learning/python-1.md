In Python, variables are created by simply assigning a value:

```python
name = "Santiago"
age = 25
height = 1.85
is_student = True
```

Python has four basic data types: strings, integers, floats, and booleans. Unlike other languages, you don't need to declare the type explicitly — Python figures it out automatically.
